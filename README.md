# 타일 기억 게임 - TileMemoryGame
기억력 향상을 위한 타일 기억 게임

## 개발 환경
+ JavaScript
+ css
+ html

## 프로젝트 소개
색이 표시된 타일을 기억하여 클릭하는 간단한 게임으로 특정 타일을 기억한다는 점에서 기억력이 향상될 수 있다고 본다.
