# 타일 기억 게임 - TileMemoryGame

## 프로젝트 소개
순간적으로 색이 표시되는 타일을 기억하여 클릭하는 게임으로, 기억력 향상에 도움을 줄 수 있다.

## 개발 환경
+ 개발 기간 : 2020.09 ~ 2020.11 (3개월)
+ 언어 : JavaScript
+ IDE : Visual Studio Code
